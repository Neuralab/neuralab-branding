=== Neuralab Branding ===
Requires at least: 6.5
Tested up to: 6.5
Stable tag: 1.1.2
Requires PHP: 8.1
License: MIT License
Adds Neuralab branding to WordPress dashboard and login.

== Installation ==
Visit [the repo](https://bitbucket.org/neuralab/neuralab-branding/).
